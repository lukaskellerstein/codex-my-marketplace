## Font Pairings

### Classic Professional
- **Headings**: Georgia (serif, bold)
- **Body**: Calibri (sans-serif, regular)
- **Use for**: Business reports, proposals, executive summaries

### Modern Clean
- **Headings**: Calibri (sans-serif, bold)
- **Body**: Calibri (sans-serif, regular)
- **Use for**: Technical docs, internal memos, modern reports

### Traditional Academic
- **Headings**: Cambria (serif, bold)
- **Body**: Cambria (serif, regular)
- **Use for**: Academic papers, research reports, formal documents

### Bold Impact
- **Headings**: Arial Black or Arial Bold (sans-serif)
- **Body**: Arial (sans-serif, regular)
- **Use for**: Marketing materials, proposals, pitches

### Elegant Serif
- **Headings**: Palatino Linotype (serif, bold)
- **Body**: Garamond (serif, regular)
- **Use for**: Luxury proposals, formal letters, annual reports

## Typography Scale

### Document Hierarchy

| Element | Font Size | Weight | Spacing Before | Spacing After |
|---------|-----------|--------|---------------|---------------|
| Title | 26-28pt (52-56) | Bold | 0 | 200 |
| Subtitle | 14-16pt (28-32) | Regular/Italic | 0 | 200 |
| Heading 1 | 18pt (36) | Bold | 360 (18pt) | 120 (6pt) |
| Heading 2 | 14pt (28) | Bold | 240 (12pt) | 80 (4pt) |
| Heading 3 | 12pt (24) | Bold | 200 (10pt) | 60 (3pt) |
| Body text | 12pt (24) | Regular | 0 | 120 (6pt) |
| Caption | 10pt (20) | Regular/Italic | 60 | 60 |
| Footnote | 9pt (18) | Regular | 0 | 0 |

### Line Spacing

| Context | Line Spacing | Value |
|---------|-------------|-------|
| Single | 1.0x | 240 |
| Body text | 1.15x | 276 |
| Comfortable reading | 1.5x | 360 |
| Double-spaced (draft) | 2.0x | 480 |

## Spacing Guidelines

### Page Margins

| Document Type | Margins | DXA |
|---------------|---------|-----|
| Standard report | 1" all sides | 1440 |
| Formal letter | 1" top/bottom, 1.25" sides | top/bottom: 1440, sides: 1800 |
| Wide tables | 0.75" sides | 1080 |
| Presentation-style | 0.5" all sides | 720 |

### Vertical Rhythm

- Between sections (Heading 1): 360 DXA before (~18pt gap)
- Between subsections: 240 DXA before (~12pt gap)
- Between paragraphs: 120 DXA after (~6pt gap)
- After tables: 200 DXA
- After images: 200 DXA

## Design Patterns

### Cover Page

Every professional document should have a cover page with:
- Document title (large, bold, primary color)
- Subtitle or description (smaller, muted color)
- Author name and date
- Optional: company logo or cover image
- Separate section with no header/footer

### Table Design

- **Header row**: Primary color background, white bold text
- **Alternating rows**: White and light-bg (e.g., F5F5F5)
- **Borders**: Light gray (CCCCCC), thin (1-2pt)
- **Cell padding**: 80-120 DXA top/bottom, 120-200 DXA left/right
- **Column widths**: Set explicitly — never rely on auto width

### Callout Boxes

Use single-cell tables with:
- Background fill (light version of accent or primary color)
- Left border emphasis (thick, primary color)
- Padding: 200 DXA top/bottom, 300 DXA left/right
- **Types**: Info (blue), Warning (amber), Success (green), Note (gray)

### Image Placement

- **Full-width images**: Center aligned, 6.5" wide
- **Text-wrap images**: Float left or right, 3" wide, with margins
- **Diagrams/charts**: Center aligned, caption below
- Always include a caption paragraph below significant images

### Color Usage Rules

- **Primary color**: Section headings, table headers, callout borders
- **Secondary color**: Subheadings, links, secondary emphasis
- **Accent color**: Highlight numbers, key metrics, CTAs
- **Light BG**: Callout backgrounds, alternating table rows
- **Text color**: All body text (never pure black — use dark gray)
- **Muted color**: Captions, footnotes, less important text

## Common Mistakes to Avoid

1. **Inconsistent heading styles** — always use the document's style system
2. **No visual hierarchy** — make headings clearly different from body text
3. **Tables without header styling** — always shade the header row
4. **Monochrome documents** — use color for headings and accents
5. **Missing page numbers** — always include page numbers in footer
6. **No Table of Contents** — add for documents > 3 pages
7. **Wall of text** — break up with headings, bullets, tables, callouts
8. **Inconsistent spacing** — use paragraph spacing, not blank lines
9. **Pure black text** — use dark gray (333333) for body, pure black is harsh
10. **All caps headings** — use size and weight for hierarchy, not ALL CAPS
